Unity – Planetary Escape

Role: Gameplay Designer (Solo) · Platform: PC

This university assignment was designed to simulate real and professional prototyping. With a focus on timeboxing, scope management and rapid iteration.
I was responsible for defining the core gameplay idea, building a prototype, having the prototype tested and iterated within a fixed timeframe.

  • Design focus: Prototyping, Testing, Iteration
  • Constraints: Fixed development time, no visual machine.
  • Results: Conducted informal playtests, Observed player engagement and pacing

Unity · Visual Studio · GitHub · C#

Unreal Engine – Merchant Troubles

Role: Gameplay Designer (Solo) · Focus: Gameplay Systems, implementation

This project was focused on how systems can be implemented using both C++ and Blueprints inside of Unreal Engine.
The goal behind this project was to connect gameplay design intent with technical implementation enabling better control over the player experience.

  • Design focus: Gameplay systems structure, iteration through implementation
  • Designed systems: Player abilities and input handling
  • Implementation: Core logic implemented through C++, blueprints used for final iteration and tweaks

Unreal Engine · Rider · C++ · Blueprints · GitHub

Unity – Parkour Master
