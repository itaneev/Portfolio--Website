Unreal Engine – Deadplate

Role: Game Designer (Group project) · Platform: PC

Cook and serve food in a fast-paced kitchen where hungry monsters are waiting. ake orders, manage ingredients, and work together to keep up as the pressure builds and things start to get chaotic.
The focus of this project was to simulate the team environment in games development. From the work done alone, to the work needed to be done with others.

Unreal Engine – Merchant Troubles

Role: Gameplay Designer (Solo) · Focus: Gameplay Systems, implementation

This project explored the implantation of gameplay systems using both C++ and Blueprints in Unreal Engine.
I designed and implemented gameplay mechanics, using Blueprints for rapid iteration and C++ for more controlled and scalable functionality.
This allowed me to better connect design intent with technical execution, improving my ability to shape player experience through both design and implementation.

  • Design focus: Gameplay systems structure, iteration through implementation
  • Designed systems: Player abilities and input handling
  • Implementation: Core logic implemented through C++, blueprints used for final iteration and tweaks

Unreal Engine · Rider · C++ · Blueprints · GitHub
